## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript
- PeerJS (WebRTC)
- jsPDF

Backend:
- Python
- Flask
- AssemblyAI API

## How to Run

1. Backend:
   cd backend
   python assembly_api.py

2. Frontend:
   Open frontend/index.html in browser
